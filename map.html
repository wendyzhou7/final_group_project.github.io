NYC WRRF Distribution Map

map <-
  read_csv("data/HydroWASTE_v10.csv")
  

wwtp <-
  read_csv("data/SARS-CoV-2_concentrations_measured_in_NYC_Wastewater_20231129.csv")

map_df <-
  map |>
  janitor::clean_names() |>
  select(waste_id, wwtp_name, lat_wwtp, lon_wwtp) |>
  mutate(wrrf_abbreviation = recode( wwtp_name,
    "New York C   Rockaway WPCP" = "RK",
    "New York C   Red Hook WPCP" = "RH",
    "New York C   Port Richmond WPCP" = "PR",
    "New York C   Oakwood Beach WPCP" = "OB",
    "New York C   26th Ward WPCP" = "26W",
    "New York C   Tallman Island WPCP" = "TI",
    "New York C   North River WPCP" = "NR",
    "New York C   Coney Island WPCP" = "CI",
    "New York C   Jamaica WPCP" = "JA",
    "New York C   Hunts Point WPCP" = "HP",
    "New York C   Owls Head WPCP" = "OH",
    "New York C   Bowery Bay WPCP" = "BB",
    "New York C   Newtown Creek WPCP" = "NC",
    "New York C   Wards Island WPCP" = "WI",
  ))

wwtp_df <-
  wwtp |>
  janitor:: clean_names()

merge_df <- inner_join(map_df, wwtp_df, by = "wrrf_abbreviation")

nyc_wastewater <-
  merge_df |>
  rename(concentration = concentration_sars_co_v_2_gene_target_n1_copies_l) |>
  drop_na(concentration) |>
  separate(sample_date, into = c("month", "day", "year"), convert = TRUE) %>% 
   mutate(
     year = as.character(year),
     month = factor(month, levels = 1:12),
     month = recode(month,
                        "1" = "January",
                        "2" = "February",
                        "3" = "March",
                        "4" = "April",
                        "5" = "May",
                        "6" = "June",
                        "7" = "July",
                        "8" = "August",
                        "9" = "September",
                        "10" = "October",
                        "11" = "November",
                        "12" = "December")) |>
  select(-waste_id, -test_date, -per_capita_sars_co_v_2_load_n1_copies_per_day_per_population, -population_served_estimated, -wwtp_name, -annotation, -wrrf_abbreviation)

wttp_map <-
  nyc_wastewater %>% 
  filter(year %in% c(2021, 2022), technology == "RT-qPCR") %>% 
  group_by(wrrf_name) %>%
  summarise(avg_conc_area = mean(concentration))
            
map_df <- left_join(nyc_wastewater, wttp_map, by = "wrrf_name")


Our objective is to generate a visual representation of SARS-CoV-2 concentrations in wastewater across various Wastewater Resource Recovery Facilities (WRRFs) in New York City.

The initial segment of the code involves data loading and cleaning. For the WRRF information data, relevant columns are selected and prepared for further analysis. Abbreviations for WRRF names are established for convenient reference. Concentration data is also loaded and cleaned, and the two datasets are merged based on the WRRF abbreviation. In the merged dataset, date information is segmented into day, month, and year columns, and the dataset is filtered to eliminate rows with missing concentration values. Unnecessary columns are dropped for simplicity.

map <-
  map_df %>% 
  leaflet() |>
  addTiles() |>
  addCircleMarkers(
             lng = ~lon_wwtp,
             lat = ~lat_wwtp,
             radius = 7,
             stroke = TRUE,
             fillOpacity = 0,
             color = "skyblue",
             popup = ~paste(sep = "<br/>",
                            "<b>", wrrf_name,
                            "</b><br>Mean Concentration:", avg_conc_area, "(N1 Copies/L)")) %>% 
  addProviderTiles(providers$CartoDB.Positron)

map


In the subsequent part of the code, a map is constructed using the Leaflet package. Initially, the average concentration for each WRRF is computed for the years 2021 and 2022, considering only data from the RT-qPCR technology. The original dataset is then left-joined with this summary data based on the WRRF name.

The Leaflet map is generated using the resulting map_df dataframe, featuring tiles from the CartoDB Positron map as a base layer. Circle markers are incorporated to denote each WRRFs geographical location, and the radius and color of the circles are adjusted for enhanced visualization. Popup information is included, showcasing the WRRF name and the mean concentration of SARS-CoV-2 (N1/L).

The resulting map offers an vivid depiction of the distribution of SARS-CoV-2 concentrations across different WRRFs in New York City, allowing for visual distinctions among areas with higher or lower concentrations, and the popup information facilitates more detailed insights into specific WRRFs.
